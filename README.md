## Endpoints

- `/` Home Page
- `/leagues` Lista de campeonatos
- `/leagues/:league` Lista de todos os times de uma determinada liga
- `/leagues/:league/team/:team` Detalhe de um selecionado time
- `/categories` Add and list categories
- `/add-video` Add video
